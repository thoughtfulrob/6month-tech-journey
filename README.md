# 6-Month Tech Journey

This repository documents my 6-month accelerated learning plan across systems administration, networking, Linux, Windows, cybersecurity fundamentals, automation, and hands-on labs.

## Structure

- **Week00–Week20** – Weekly labs and notes following a structured curriculum:
  - Networking fundamentals
  - Linux & Windows administration
  - Security fundamentals & SIEM
  - Python automation
  - Cloud basics
- **Projects/** – Larger portfolio pieces that span multiple weeks.
- **Scripts/** – Python/PowerShell and other automation utilities.
- **Configs/** – VM setups, tool configuration files, environment notes.
- **Notes/** – Concept summaries, cheat-sheets, and reusable reference material.

## Goals

- Build a job-ready foundation for IT and cybersecurity roles.
- Practice troubleshooting across Windows, Linux, networking, and security tools.
- Develop small but real automation scripts and labs.
- Maintain an audit trail of learning for interviews and personal reflection.

## How to Read This Repo

- Start with the **WeekXX/** folders to see what I worked on each week.
- Check **Projects/** for deeper, more polished work.
- Look at **Notes/** for how I understand and summarize key concepts.
- Explore **Scripts/** to see the kinds of tasks I automate.
